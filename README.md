# Aura At Çiftliği

Aura At Çiftliği için statik web sitesi.

## Sayfalar

- `index.html` — Anasayfa
- `hakkimda.html` — Hakkında
- `iletisim.html` — İletişim

## Yerel çalıştırma

```bash
npx serve
```

Ardından tarayıcıda `http://localhost:3000` adresini açın.

## GitHub Pages

Site `main` dalından otomatik yayınlanır.

Canlı adres: GitHub deposu ayarlarından Pages URL'sini kontrol edin.
